# Healthcare-Insurance-Big-Data-Project



\# Healthcare Insurance Big Data Project



\## Project Overview



This capstone project focuses on building a Big Data solution for a healthcare insurance company. The goal is to analyze claims, subscribers, patients, policy groups, hospitals, and premium-related data to support business decision-making and revenue improvement.



\## Technologies Used



\* AWS S3

\* AWS Redshift

\* Databricks

\* AWS EMR Studio

\* PySpark

\* Jira

\* GitHub



\## Project Documentation



The documentation files are available inside the `documents` folder:



1\. Requirement Specification Document

2\. Solution Design Document

3\. Database Schema Design Document



\## Project Goal



The goal of this project is to create data pipelines that clean, process, and analyze healthcare insurance data. The final outputs will help the company understand customer behavior, claim patterns, hospital usage, rejected claims, profitable policy groups, and premium-related insights.
